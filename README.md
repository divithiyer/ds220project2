# Gold Price Analysis (1833–2026)

##  Project Overview
This project analyzes historical gold price data from 1833 to 2026 using Python and Pandas. The goal is to explore long-term trends, volatility, growth patterns, and key insights from nearly 200 years of gold price data.



##  Dataset
The dataset used is **gold_advanced_features.csv**, which includes:

- Monthly gold prices (USD per ounce)
- Date-based features (year, month, quarter)
- Engineered features such as:
  - Rolling averages
  - Lag values
  - Momentum indicators
  - Percentage changes
  - Volatility measures



##  Questions Addressed

1. How has gold price changed over time?
2. Which periods saw the highest growth in gold prices?
3. What are the average gold prices by decade?
4. How volatile are gold prices over time?
5. Are there noticeable trends or cycles in gold prices?
6. What are the highest and lowest price periods?



##  Analysis Performed

- Time-series visualization of gold prices
- Growth rate analysis using percentage change
- Decade-wise average price calculation
- Volatility analysis using rolling standard deviation
- Trend identification using rolling averages
- Identification of highest and lowest price periods



##  Key Insights

- Gold prices show a strong long-term upward trend.
- Growth and volatility increase significantly in recent decades.
- The most rapid price increases occur during periods of economic uncertainty.
- Gold has transitioned from a stable commodity to a dynamic financial asset.
- The highest price occurs in 2026, while the lowest is observed in earlier historical periods


##  Tools & Libraries Used

- NumPy
